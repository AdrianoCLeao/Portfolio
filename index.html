Projects

Notable projects I've delivered.

Machine Learning is not just about building a model.
My work focuses on delivering results and ensuring reliability. A great model is useless if it remains confined to a notebook. Each project is carefully designed to solve real-life problems and pave the way for the future.

Galactic Atlas

Using K-Nearest Neighbors algorithm to identify exoplanets using data collected by Kepler Space Telescope. Despite challenges such as a big class imbalance and sensitivity to outliers, the model became highly accurate.

BirdSense

The Bird Sense Project uses neural networks to develop an app for identifying birds from photos. The app analyzes images and accurately recognizes over 500 species, utilizing Keras from TensorFlow for its neural network implementation.
